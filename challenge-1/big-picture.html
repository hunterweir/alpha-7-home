Can This Process Save the World?

I think it can. Imagine deliberate practice around the worlds thorniest issues!

As an example, think: applied systems theory. Or ecological economics. Or permaculture.

These obviously need to be fleshed out quite a bit, but the idea is that breaking big, hairy audacious goals down into manageable chunks, we can make magic.

The cool part is this is a meta skill, so you can learn it in one domain and apply it elsewhere.
